은평 와와코칭센터
반복과 정리, 설명과 시각화를 적절히 조합하면서 학습은 더 이상 산발적인 정보의 나열이 아니라, 서로 연결된 하나의 고정된 구조물이 되어 단단히 자리 잡게 된다. 특히 초등학교 6학년처럼 기초는 탄탄하지만 응용에서 흔들리는 시기에 이러한 전환은 단순한 학습법의 변화가 아니라 사고의 설계도를 바꾸는 기회가 되며, 이 시기를 얼마나 깊이 있고 체계적으로 헤쳐 나가느냐가 중등 교육의 성공 여부를 가르는 분기점이 된다. 은평 와와코칭센터은 개념이 익숙해져도 응용 상황에서 막히는 학생은 흔히 필기는 꼼꼼하지만 사고의 확장력이 부족한 특징을 가지며 이들에게는 문제 풀이 전에 이 문제는 어떤 개념을 묻고 있는가를 반드시 적게 유도하는 훈련이 필요하다. 문제를 풀기 전에 파란색 형광펜으로 문제에서 주어진 조건을, 빨간색으로는 질문에서 요구하는 것을 밑줄 치게 하며, 교재의 답은 잘 보지만 문제의 지문을 제대로 읽지 않는 패턴을 점차 개선한다. 은평 와와코칭센터은 예를 들어, ‘환경 문제’라는 단어 하나를 중심으로 신문 기사, 과학 보고서, 문학 작품 속 서술, 논설문까지 비교하며 각각 어떻게 접근하는지를 분석한다. 학습 효율을 높이기 위해 개념 정리용 종이를 작게 사용하고, 풀이 순서 체크리스트를 도입하여 평균 16점 상승 효과를 경험한다. 이런 상황에서 과제는 잘 수행하지만 자기주도성이 부족한 아들은, 공부에 들어가기 전 1분 동안 목표를 마음속으로 구체화하고, 문제 풀이 과정을 기록함으로써 자신의 학습 흐름을 눈에 보이게 만들 때 비로소 학습 효율이 눈에 띄게 향상된다.
